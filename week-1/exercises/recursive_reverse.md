# Recursive Reverse

Write a function `reverse(string)` that takes in a string and returns it reversed using recursion.

```ruby
  # Test Cases
  reverse("house") # => "esuoh"
  reverse("dog") # => "god"
  reverse("atom") # => "mota"
  reverse("q") # => "q"
  reverse("id") # => "di"
  reverse("") # => ""
```
